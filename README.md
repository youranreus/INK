##说明
为了更好地使用头部
请设置以下页面
 - 名称：友链（这个随意）缩略名：links
 - 名称：关于（这个随意）缩略名：About
 - 名称：归档（这个随意）缩略名：arch

然后支持伪静态的请把链接分别设置为
links.html   about.html   arch.html
不支持的请到header.php修改相应路径
然后就可以愉快地玩耍啦
